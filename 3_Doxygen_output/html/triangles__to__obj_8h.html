Converts a list of triangles into Wavefront .obj format. Returns 3 strings corresponding respectively to the vertices, the uv_coordinates and the faces of a vector of Triangle in Wavefront .obj format.

Parameters
[in]trianglesVector containing the Triangles to convert.
[in]index_offsetIndex offset when converting several object to .obj. Default 0.
[in]x_centroidThe X coordinate of the centroid. Default 0.
[in]y_centroidThe Y coordinate of the centroid. Default 0.
Returns
A vector of 3 strings: 1-vertices 2-uv coordinates 3-faces
